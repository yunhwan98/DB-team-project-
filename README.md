- 데이터베이스에 포함된 모든 테이블에 대한 입력/삭제/변경 기능
- 관리자 기능(모든 데이터 초기화, 입력, 삭제, 변경 가능 기능)
- 학생 기능(수강 과목 조회 기능,수강 과목 시간표 기능,본인의 성적표를 보여주는 기능)
